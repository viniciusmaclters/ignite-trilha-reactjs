## 🚀 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- ReactJs
- JavaScript
- HTML
- CSS
- Vite
- date-fns
- phosphor-react

## 💻 Projeto

Consiste em uma aplicação de feed de comentários, podendo adicionar um comentário, aplaudir(curtir) e excluir o comentário. Contém as seguintes seções: Header, Sidebar, Post e Comment.
